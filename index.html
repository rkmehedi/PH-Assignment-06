Frequently Asked Questions

1. The difference between var, let, and const
const value can't be reassign but allow object property change, in let we can reassign value, var is old can be reassigned or re-declare or updated but have some problem.
2. The difference between map(), forEach(), and filter()
Map always return new array, foreach do not return anything,filter return array with the result passed by the specific filter.
3. Explain arrow functions and how they are different from regular functions
Arrow function is shorter and can return value directly if it single line code if multiline then we must give manual return and it can't be use as constructor.
4. How JavaScript Promises work
When we call it then it can give only 3 things which are pending (still working), fulfilled (resolved), or rejected(when failed).
5. How closures work in JavaScript
It is something like block system when in function it can get all globally or parent things in child but can't pass things out of own brackets.
